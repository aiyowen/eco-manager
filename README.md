# 桌面整理助手

一个简单实用的桌面整理小程序，帮助你自动分类和整理桌面文件，让桌面保持整洁。

## 功能特点

- **自动分类**：将桌面文件分为程序、文档和文件夹三类
- **按日期排序**：文件按修改日期排序，最新的文件排在前面
- **一键整理**：通过GUI界面一键完成整理操作
- **定时整理**：支持设置每天定时自动整理
- **文件重名处理**：自动为重复文件名添加时间戳

## 安装使用

### 环境要求

- Python 3.6+
- 所需依赖包：
  - schedule (用于定时任务)
  - tkinter (Python内置，无需单独安装)

### 安装依赖

```bash
pip install schedule
```

### 运行程序

1. 克隆或下载本项目到本地
2. 安装依赖包
3. 运行 `desktop_organizer.py` 文件：

```bash
python desktop_organizer.py
```

## 使用说明

1. **立即整理**：点击"立即整理"按钮，程序会立即开始整理桌面文件
2. **定时设置**：在"定时整理时间"输入框中设置每天的整理时间，格式为"HH:MM"，然后点击"更新"按钮
3. **状态显示**：界面会显示当前程序状态和整理结果

## 整理规则

- **程序**：包括 .exe、.app、.dmg 等可执行文件
- **文档**：包括 .doc、.docx、.pdf、.txt、.xls、.xlsx、.ppt、.pptx 等文档文件，以及 .zip、.rar、.7z、.tar、.gz 等压缩文件，.jpg、.jpeg、.png、.gif、.bmp 等图片文件
- **文件夹**：所有文件夹和 .lnk 快捷方式文件会被统一整理到"folders"目录
- **特殊规则**：run.bat 文件会保留在桌面上，不做整理

## 注意事项

1. 程序会在桌面创建一个名为"整理"的文件夹，用于存放分类后的文件
2. 整理过程中会移动文件，请注意备份重要文件
3. 定时任务会在后台运行，程序需要保持开启状态

## 自定义配置

可以修改 `CONFIG` 字典来自定义以下参数：

- `desktop_path`：桌面路径
- `categories`：文件分类规则
- `organized_folder`：整理文件夹路径
- `schedule_time`：默认定时整理时间

## 示例

运行程序后，桌面文件会被整理到以下结构：

```
└── 整理/
    ├── programs/     # 程序文件
    ├── documents/    # 文档文件
    └── folders/      # 文件夹
```
